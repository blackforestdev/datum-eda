Datum·EDA · Title Block · Size Matrix

One block, every size — fixed & corner-anchored

The block is a constant physical size, pinned to the corner. Letter, Tabloid (11×17), and ANSI-D (24×36) all carry the same block — the drawing area grows, the block doesn't. The ratios and the Grauwert tone are size-invariant, so nothing is redrawn per size; this is the §8 guideline projected. Sheets shown nested at a common scale.
