# Datum EDA Glyph Set

These SVGs are the first custom EDA glyphs required by
`docs/gui/VISUAL_LANGUAGE.md` §4. They are Datum-owned line icons on a 24px
keyline grid, designed to sit beside Tabler-style chrome icons.

Rules:
- `viewBox="0 0 24 24"`
- monochrome strokes using `currentColor`
- no hardcoded fill color
- round line joins/caps unless the geometry requires a sharp marker
- source SVGs are the editable asset of record until an icon-font/MSDF pipeline
  lands
